# Js-Slot Machine (JavaScript)

A simple slot machine game built using JavaScript. The game runs in the console and simulates basic slot machine behavior using random outcomes.

## Features

- Random symbol generation
- Win and loss calculation
- Balance tracking across rounds
- Simple game loop with user input

## Tech Stack

- JavaScript (Node.js)

## How It Works

The player starts with a balance and places a bet.  
Each spin generates random symbols.  
If the symbols match based on predefined rules, the player wins.  
The balance updates after every round until the player quits or runs out of money.

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the game using Node.js:

```bash
node index.js
